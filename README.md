#Fishing idiots

## サイト概要
釣り人が釣った魚の写真を投稿できるサイトです。

### サイトテーマ
釣って楽しい、（記録を）残して楽しい。

### テーマを選んだ理由
自分の趣味が釣りでいろんな魚を釣るのですが釣った魚の画像や、 調理した画像を共有したいと思いこのサイトを制作しました。

### ターゲットユーザ
釣り人・魚を調理したい人

### 主な利用シーン
自分の釣った魚をシェアして写真やコメントをシェアしたい時
